Introduction

J’ai effectué mon stage au sein de la société Tunvita du 5 juillet au 1 aout 2021.

Dans ce site je vais vous présenter au premier lieux la sociéte puis les taches éffectuées a moi durant ce stage

Présentation de la société

Tunvita société tunisienne de développement informatique : création site web (site vitrine, site E-commerce, site dynamique ...), solutions informatiques (finance, médical, gestion...), réseau (installation des systèmes alarme - caméra...), design

Site Web et design

-Un site internet dynamique est un site dont les pages peuvent être générées dynamiquement, soit à la demande .
-Un site vitrine est une carte de visite présentera des informations brèves sur votre société (ou association, ou organisation,…) .
-Un site E-commerce consiste à utiliser des systèmes de boutiques en ligne dans l’objectif de faire du commerce sur internet.

Solutions Informatiques

Un logiciel est par définition une solution standard mutualisée, offrant un panel de fonctionnalités pour répondre aux besoins d'utilisateur dans plusieurs domaines (finance, médical, gestion...)

Contexte du stage

Dans l’objectif de maitriser et approfondir mes connaissances déjà acquises, la société Tunvita m’a accueilli dans son locale.

Elle m'a proposé comme sujet de développer un site web qui présente une application web (Logiciel de facturation) déjà utilisé par la société en utilisant HTML CSS JS , puis de suivre des formations sur ES6 & SQL & PHP pour connaître les différentes fonctionnalités de ces outils, bien maîtriser ses connaissances à l'aide des exemples et des travaux pratiques .

Activités du stage

- Première Activité (partie front-end) :

1) Site web sans Template :
* Architecture du site :

Avant de passer à la phase de realisation du site j'ai fait son architecture

*conception et prototypage du site :

Aprés avoir terminer l'architecture ,j'ai passé à la partie conception et prototypage du site avec Adobe XD

* Développement du site :

En suite J'ai passé au développement de mon site web en appliquant mes connaissances déjà acquises en HTML , CSS et javascript.

* Hébergement sur GitHub :

Après avoir installé le GitHub sur mon ordinateur j'ai déposé les fichiers du site et à chaque fois je modifie le code je charge (Push) les fichiers de nouveau.

* Hébergement sur Netlify :

Grace à la liaison entre Netlify et GitHub ,Netlify devient un environnement de développement continu et se recompile à chaque push git

L'hébergement sur Netlify m'a permet d'activé le formulaire du contact de mon site par l’envoi d’une copie des données rempli par le visiteur sur l’email du service de Tunvita et d’enregistrer une copie sa base de données.

2) Site web avec Template :

Adapter un site avec template a mon site était la deuxième taches affecté à moi durant ce stage

- Deuxième Activité (partie Back-end) :

1) Activation du formulaire avec PHP :

En utilisant la bibliothèque PHPmailer , je me suis arrivé à envoyer une copie des informations soumis par le visiteur du site sur l’email du service de Tunvita et de lui envoyer une réponse de réception de son mail.

Page de contact

Cette page contient les coordonnées de société Tunvita et un formulaire de contact qui permet au utilisateur de contacter la société.

Message de remerciement

Cette page contient un message afin de remercirer l'utilisateur d'avoir contacter Tunvita

2) Base de données :

L'utilisation du PHP et phpMyAdmin permettre d’enregistrer une autre copie des informations soumises dans la base de données.

Base de données phpMyAdmin

A chaque fois un utilisateur rempli le formulaire du contact tous les données sont automatiquement enregistrer dans la base de données phpMyAdmin.

Conclution Générale

Le Stage que j’ai effectué au sein du Tunvita, d’une durée d’un mois, était une expérience fructueuse vue que c’est mon premier contact avec le monde professionnel.

Au cours de cette période, j’ai affronté aux vraies difficultés de travail mais avant tout j'ai m’intégré dans des groupes de personnes et travaillé en collaboration avec eux et aussi de se familiariser avec cet environnement .
